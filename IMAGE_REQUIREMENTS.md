## Image Optimization Tips

1. **Format:**
   - Use JPG for photos
   - Use PNG for logos/graphics with transparency
   - Consider WebP for better compression

2. **Compression:**
   - Use tools like TinyPNG, ImageOptim, or Squoosh
   - Target: 100-300KB per image
   - Don't sacrifice too much quality

3. **Naming:**
   - Use lowercase
   - Use hyphens, not spaces
   - Be descriptive: `jute-tote-bag-natural.jpg`

4. **Alt Text:**
   - Already configured in code
   - Describes the image content for SEO and accessibility
